# EBA-Einheiten verstehen — für KT1

**Beispiel:** `1.1.1 — Sich im Lehrvertrag orientieren` (liegt als Entwurf auf der Plattform, nur für KT1 sichtbar).

## In einem Satz

Eine EBA-Einheit ist **dieselbe Idee wie eine 3er-Einheit**, didaktisch heruntergebrochen auf die
EBA-Zielgruppe (A2, niederschwellig, viel Scaffolding) — mit **zwei** statt drei Herausforderungen und
**einem eigenen Wissens-Dossier**, weil es für die EBA kein Lehrmittel gibt.

## Was anders ist (und warum)

Backward Design, Prinzip-First, ICH-/Sie-Form, die 8 Merkmale, bi-dimensionale Bewertung und die
ganze Plattform/Bedienung bleiben **gleich**. Der Grund für jeden Unterschied ist die **Zielgruppe**:
schulisch schwächer, Sprachniveau **maximal A2**.

| Stellschraube | 3er-Set (EFZ) | EBA-Einheit | Warum |
|---|---|---|---|
| Herausforderungen | 3 (A/B/C) | **2 (A/B)** | weniger Last; zwei Fälle genügen für ein Muster |
| Spiralen-Regel | A/B/C variieren | **A und B: gleicher Trade-off, maximal verschiedene Kontexte** | aus zwei Fällen wird ein übertragbares Prinzip |
| Niveau (Bloom) | bis K4 | **K2–K3** (K4 nur als Vertiefung) | erreichbares Anspruchsniveau |
| Quoten pro Thema | 3 SK / 3 Aspekte / 3 Modi | **2 SK / 2 Aspekte / 1 Modus** | fokussierter (RLP-Minimum) |
| Kompetenznachweis | drei gleichrangige Formen | **Fachgespräch als Standard** (mündlich); Mini Case + Werkschau vereinfacht | mündlich entlastet schwache Schreiber |
| Sprache (SuK-Kriterien) | Fachkorrektheit + Argumentation inkl. Normen | gleiche Namen, aber **Konventionen + Sprachbewusstheit, KEINE Normen** | RLP-Vorgabe für EBA |
| Wochen-Raster | fix | **kein fixer Wert** — LP bestimmt das Tempo | EBA-Klassen sind sehr heterogen |
| Sprache durchgehend | — | **A2 hart erzwungen** | Lesbarkeit |
| Schlussarbeit / -prüfung | Teil des EFZ-Systems | **gibt es nicht** — der KN trägt alles | RLP |
| Layout | Standard | grössere Schrift, kurze Zeilen, eine Sache pro Block, Scaffolding sichtbar | kognitive Last raus, nicht infantil |

## Das wirklich Neue: das Wissens-Dossier

Die EBA hat **kein Lehrmittel**. Statt auf Buchseiten verweisen die Leitfragen auf ein eigenes
**Wissens-Dossier** (ein „Glossar+"): kurze Wissens-Nuggets (A-01 … B-03), je an eine Leitfrage
gekoppelt, dazu Sprachhilfen, ein Transfer-Blatt und ein A2-Glossar. Man liest es **nicht von vorne**,
man springt über Anker hinein.

Neu dazugekommen ist ein **Recherche- und KI-Layer**: pro Nugget gibt es fertige Suchbegriffe, einen
A2-gerechten KI-Prompt (immer mit dem Hinweis, die Antwort an einer sicheren Quelle zu prüfen),
KI-Lernstrategien und einen Selbst-Prüf-Auftrag. So lernen EBA-Lernende, Wissen **selbst zu finden und
zu prüfen** — die EBA-Antwort auf das fehlende Lehrmittel.

*Hinweis fürs Review: kantonsabhängige Fakten (z. B. Name der Kontaktstelle) sind als „LP prüfen"
markiert; die übrigen Rechtsangaben sind validiert (OR / berufsbildung.ch).*
